## Clone repository

``` bash
git clone -v https://github.com/AbelPhilippe/plcharger
```
```bash
cd plcharger
```

## Use VENV

```bash
python3 -m venv venv
```
```bash
source venv/bin/activate
```

## Install requirements.txt

```bash
pip install -r requirements.txt
```

## Execute TBUG (UBUNTU / DEBIAN / KALI / WSL)

```bash
python plcharger.py -i payload.bin -png carrier.png -o output.png
```
